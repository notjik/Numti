Numti v1.0.4

<center><h1>Hello, my friend!</h1></center>
<h3>With this program, you can use contact recording not only on your phone, but also on your PC.</h3>
<h4>First you need to install the program.</h4>
<ol><strong>
 <li>Open "setuper" directory.</li>
 <li>Use "numti_setup.exe" following the instructions inside it.</li>
 <li>Enjoy!</li>
</strong></ol>
<h4>Else you can use the program directly inside the downloaded directory.</h4>
<ol><strong>
 <li>Open "dist" directory.</li>
 <li>Use "Numti.exe".</li>
 <li>Enjoy!</li>
</strong></ol>
<h4>In the program you can:</h4>
<ul><strong>
  <li>View your contacts</li>
  <li>Add/Edit/Delete contacts.</li>
  <li>Search for numbers from a document or search bar.</li>
</strong></ul>
